Project Description

This C++ simulation models the trajectory of a projectile launched from a howitzer. Built as a collaborative partner project, it leverages real-world physics and object-oriented design to calculate and visualize artillery motion with high fidelity.

The Physics class dynamically adjusts gravitational acceleration based on the projectileâ€™s altitude, simulating how gravity weakens at higher elevations. The Projectile class handles the core motion calculations, applying velocity updates over time. The Howitzer class encapsulates initial launch logic such as angle, velocity, and position.

The simulation is designed to be abstract, robust, and maintainable. Special attention was paid to keeping modules isolated, testable, and logically cohesive. We used test-driven development (TDD) throughout the project to validate components, improve stability, and ensure future-proof extensibility.

The program was developed in C++ using modern programming principles. Its modular design supports future upgrades such as air resistance, terrain interaction, or visual GUIs.

This project was completed in Fall 2025 in an advanced C++ course using TDD methodology and collaboration.
